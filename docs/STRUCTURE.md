# RPi4 Communications Repo – Structure

- i2c/        — I2C test scripts, config, notes
- spi/        — SPI test scripts, overlays, config
- uart/       — Serial console & UART notes and tools
- can/        — MCP2515, SocketCAN, CAN tests
- rs485/      — RS485 adapter usage and tests
- gpio/       — Simple GPIO examples, pin maps
- examples/   — Combined / higher-level examples
- docs/       — General documentation
- docs/datasheets/ — Links or important datasheets
